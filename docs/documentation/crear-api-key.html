Usando API Keys

Esta guía muestra cómo crear un API KEY.

Creando API keys

La API key es un identificador único que autentica las solicitudes asociadas con su aplicación para fines de uso y facturación. Debe tener al menos una API key asociada con su aplicación.

Para crear un API Key:

  1. Iniciar sesión en la Consola de administración. Por favor cree una cuenta si no la tiene.
  2. En el menú izquierdo seleccione la opción Crear API Key.
  3. Haga click en el botón Crear API Key y copie el resultado para su posterior uso. Este valor no puede ser consultado posteriormente.
