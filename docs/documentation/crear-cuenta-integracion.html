Crear cuentas de integración

Para poder publicar propiedades primero debe registrar las cuentas de los portales inmobiliarios, esta información es necesaria para que nuestro sistema pueda autenticarse y enviar la información de propiedades por usted.

Crear cuenta

El siguiente es un ejemplo de como crear una cuenta. 

        
          curl -X POST-H 'Authorization: Bearer $API_KEY' -H "Content-Type: application/json" -d

         '
          {
            "is_active": true,
            "emails": "contact1@myrealstateagency.com,contact2@myrealstateagency.com",
            "phones": "5773365,668899",
            "cellphones": "3548889922,3214447889",
            "whatsapp_number": "3548889922,3214447889",
            "auth_user": "myuser",
            "auth_password": "mypass",
            "integration_websites": 1
          }
         '
          
          {{site.api_url}}/integration_accounts_websites

Atributos

Cuando se publica una propiedad algunos de los atributos son obligatorios y otros son opcionales, a continuación encontrará la lista de todos los atributos disponibles para crear una cuenta.

ID Descripción Requerido
is_active Se usa para indicar si la cuenta está activa. True
emails El o los emails se enviaran como datos de contacto de las propiedades si no se especifican en la publicación. True
phones El o los telefonos se enviaran como datos de contacto de las propiedades si no se especifican en la publicación. True
cellphones El o los telefonos celular se enviaran como datos de contacto de las propiedades si no se especifican en la publicación. True
whatsapp_number El o los numeros de whatsapp se enviaran como datos de contacto de las propiedades si no se especifican en la publicación. True
auth_user Usuario con el que inicia sesión en el portal. True, para Metrocuadrado y Fincaraiz
auth_password Contraseña con el que inicia sesión en el portal. True, para Metrocuadrado y Fincaraiz
integration_websites Id del portal, consulte aqui los id de los portales disponibles.. True
