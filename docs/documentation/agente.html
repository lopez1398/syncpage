Insertar agente y actualizarlo

Para poder publicar propiedades a metrocuadrado si desea enviar el agente primero debera agregarlo.

Consultar agentes metrocuadrado

El siguiente es un ejemplo de como consultar agente. 

        
          curl -X GET-H 'Authorization: Bearer $API_KEY' -H "Content-Type: application/json" -d
          
          {{site.api_url}}/metrocuadrado_agents/integrations_accounts/{id_cuenta}

Agregar agente

El siguiente es un ejemplo de como agregar un agente a la cuenta. 

        
          curl -X POST-H 'Authorization: Bearer $API_KEY' -H "Content-Type: application/json" -d
          
        '
        {
            "integration_account_websites":6,
            "operation_type": 1,
            "agent_id": 341727
        }
        '

          {{site.api_url}}/metrocuadrado_agents/integrations_accounts

Atributos

Se puede agregar un agent_id por operation_type, con este mismo metodo se puede actualizar el agent_id.

ID Descripción Requerido
integration_account_websites El id de la cuenta, consulte aqui los id de las cuentas disponibles.. True
operation_type El tipo de operacion, consulte aqui los tipos de operación. True
agent_id El agent_id de metrocuadrado. True
