Atributos

Cuando se publica una propiedad algunos de los atributos son obligatorios y otros son opcionales, a continuación encontrará la lista de todos los atributos disponibles para publicar una propiedad.

ID Descripción Requerido
integrate_to.integrations_accounts[$i].id Id de la cuenta a la cual desea integrar. Esta debe haber sido creada previamente, puede consultar la documentación de como crear una o varias cuentas aquí. True
integrate_to.integrations_accounts[$i].contact_info Información de contacto, si no se envia, se usará la información que se configuró cuando se creó la cuenta. False
property.code Código interno de la propiedad. Longitud máxima =20 True
property.title Longitud máxima =140. True
property.description Longitud máxima =2000. True
property.city Ciudad de la propiedad, consulte aqui los codigos de ciudad disponibles. True
property.zone Zona de la propiedad. Longitud máxima = 100 True
property.latitude True
property.longitude True
property.longitude Zona de la propiedad. Longitud máxima = 100 True
property.address Algunos portales utilizan la dirección para mejorar la posición del mismo en las búsquedas, por este motivo recomendamos enviarla. False
property.address_complement Complemento de la dirección. False
property.postal_code True
property.neighborhood Barrio de la propiedad. Longitud máxima = 45. True
property.url_video False
property.url_tour_3D False
property.postal_code Zona de la propiedad. Longitud máxima = 100. False
property.important Destacado. False
property.rooms True
property.bathrooms True
property.half_bath Medios baños. True
property.parking True
property.build_year Año de construcción. True
property.property_type Tipo de propiedad, consulte aqui los tipo de propiedades disponibles. True
property.conservation Conservación, consulte aqui las conservaciones disponibles.. True
property.build_year Año de construcción. True
property.pictures Fotos de la propiedad, minimo = 1, maximo = 30. True
property.pictures.name Nombre de la foto, Longitud máxima = 30. True
property.pictures.url True
property.pictures.thumb_url True
property.pictures.type PIC = Foto, PIC_360 = Foto 360, PLANE = Foto de los planos. True
property.additional_prices.mean_mantience Promedio de gastos para servicios públicos. Este Campo puede enviarse en cero. True
property.additional_prices.administration Este Campo puede enviarse en cero. True
property.additional_prices.currency consulte aqui los tipo de monedas. True
property.operation_price.price True
property.operation_price.operation_type consulte aqui los tipos de operación. True
property.operation_price.currency consulte aqui los tipo de monedas. True
property.comments False
property.measurements Se debe enviar por lo menos el área lote. True
property.measurements.measurement consulte aqui los tipos de medidas. True
property.measurements.unitmeasurement consulte aqui las unidades de medida. True
property.measurements.value consulte aqui las unidades de medida. True
property.amenities consulte aqui las características disponibles por tipo de inmueble, Cambie el parámetro property_type_id por un identificador válido. consulte aqui los tipo de propiedades disponibles. True
