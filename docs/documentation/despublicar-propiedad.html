Despublicar en todas las cuentas

Si desea despublicar la propiedad en todas las cuentas, debe enviar la etiqueta integrations_accounts vacía.

      
        curl -X DELETE -H 'Authorization: api-key $API_KEY' -H "Content-Type: application/json" -d

        '{
          "integrate_to":{
            "integrations_accounts": []
            
          },
          "property":{
              "code": "code125",
              "title": "Casa en venta zona norte barrio bello lindo.",
              ...
          }
          {{site.api_url}}/properties/{property_id}
